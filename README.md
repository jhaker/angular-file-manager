# File Explorer in Angular 6

## Installation
####install with npm
```console
npm install 
```
####install with yarn
```console
yarn install
```

## Running
####run with npm
```console
npm serve 
```
####run with yarn
```console
yarn start
```
optional port 
npm serve --port 8080
yarn start --port 8080

####open browser 
http://localhost:4200
